# Quantum Random Number Generatot

## Setup

```sh
# create a venv
python -m venv venv

# activate venv
source .../qrng/venv/bin/activate.fish

# install deps
python -m pip install -r requirements.txt

# add `/opt/advantech/examples/Python/Automation` to PYTHONPATH:
ln -fs /opt/advantech/examples/Python/Automation ./venv//lib/python3.10/site-packages/Automation
```
